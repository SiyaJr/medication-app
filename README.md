# Medication Management Web App

A web-based medication tracker built with:
- Flask (Python)
- MySQL (Database)
- Bootstrap 5 (Frontend)
- APScheduler (for reminders)

## Features
- User Registration & Login
- Medication Tracker with Pill Count
- Email Reminders
- Statistics Dashboard
- CSV/PDF Export
- Dark Mode Toggle
